# Glossário de Termos

> Direito Digital

## Instalação

Instale as dependências do projeto com o seguinte comando:

```sh
npm install
```

## Como rodar o projeto

Rode o frontend com o seguinte comando:

```sh
npm run dev
```

## Tecnologias usadas

Para implementação do frontend foi usado:

1. Typescript
2. Vite
3. React
4. Tailwind
